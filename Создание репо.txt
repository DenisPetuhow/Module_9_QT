Создание репозитория из уже существующей локальной папки — стандартная задача. Вот пошаговая инструкция.

Предполагается, что Git у вас уже установлен (если нет — скачайте с git-scm.com).

Шаг 1. Создайте пустой репозиторий на GitHub
Зайдите на GitHub.

Нажмите «+» в правом верхнем углу -> New repository.

Введите имя репозитория.

Важно: Не ставьте галочки напротив "Add a README file", ".gitignore" или "License". Репозиторий должен быть полностью пустым, чтобы мы могли загрузить туда вашу папку без конфликтов.

Нажмите Create repository.

Шаг 2. Подготовьте локальную папку
Откройте папку с вашим проектом в Проводнике.

Нажмите правой кнопкой мыши в пустом месте папки и выберите "Open Git Bash here" (или "Open in Terminal", если у вас Windows 11).

В открывшейся консоли введите команду инициализации:

Bash

git init
(Это создаст скрытую папку .git, теперь папка отслеживается гитом).

Шаг 3. Добавьте файл .gitignore (Очень важно для Qt!)
Перед тем как загружать файлы, нужно исключить мусор (папки сборки build-..., файлы .o, .exe и т.д.), чтобы они не засоряли репозиторий.

Создайте в папке проекта текстовый файл с названием .gitignore (точка в начале обязательна).

Откройте его в блокноте и вставьте туда этот текст (стандарт для Qt/C++):

Plaintext

# C++ objects and libs
*.slo
*.lo
*.o
*.a
*.la
*.lai
*.so
*.dll
*.dylib

# Qt-specific
*.user
*.qmake.stash
*.qmake.cache
Makefile*
build-*/
debug/
release/
*.pro.user
Сохраните файл.

Шаг 4. Сохраните изменения (Коммит)
Теперь вернитесь в терминал (консоль) и выполните команды по очереди:

Добавляем все файлы (кроме тех, что в .gitignore) в индекс:

Bash

git add .
Делаем первый "снимок" (коммит) состояния:

Bash

git commit -m "Initial commit"
Переименовываем ветку в main (современный стандарт):

Bash

git branch -M main
Шаг 5. Свяжите с GitHub и отправьте
На странице вашего пустого репозитория на GitHub (которую мы создали в Шаге 1) будет блок кода с заголовком …or push an existing repository from the command line.

Скопируйте команду, которая начинается на git remote add origin .... Она выглядит примерно так:

Bash

git remote add origin https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
Вставьте её в терминал и нажмите Enter.

Отправьте файлы на сервер:

Bash

git push -u origin main
После этого Git может попросить вас войти в аккаунт (откроется браузер или окошко для ввода токена). После успешного входа ваши файлы появятся на сайте.
